# Manikandan N — Portfolio

**Technical Support Engineer** | Bengaluru, Karnataka, India

A modern, premium single-page portfolio showcasing professional experience, certifications, and technical competencies.

## 🔗 Live Site

[manikandann95.github.io/Portfolio](https://manikandann95.github.io/Portfolio/)

## ✨ Features

- 🌙 Dark / Light theme toggle with localStorage persistence
- ✨ Smooth scroll animations (AOS — Animate On Scroll)
- 📱 Fully responsive design with mobile hamburger menu
- ⌨️ Typing animation for hero subtitle
- 📊 Animated stat counters
- 🎨 Glassmorphism navbar with backdrop blur
- 🌟 Particle background effects
- 💼 Timeline-based experience section

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom Properties, Glassmorphism, Grid, Flexbox)
- Vanilla JavaScript
- [Font Awesome 6](https://fontawesome.com/) — Icons
- [AOS](https://michalsnik.github.io/aos/) — Scroll Animations
- [Google Fonts (Outfit)](https://fonts.google.com/specimen/Outfit) — Typography
